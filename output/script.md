# 站点管理功能演示视频脚本

## 1. 视频概述

- **目标受众**: 企业IT管理员、视频会议系统运维人员、多地区部署的企业决策者
- **核心信息**: 智能站点管理，让每个地区的会议都快人一步，资源压力不再集中
- **情感基调**: 专业、高效、解决问题导向
- **总时长**: 75 秒

---

## 2. 分镜脚本

**时长**: 0-5秒

## 4. 关键要素清单

### Hook（前10秒）
"多地区部署时，所有终端都挤进总部服务器...视频延迟高、总部压力大、录制文件到处找"

### 核心卖点（优先级排序）
1. **就近接入，延迟降低93%** - 用户最关心的性能问题
2. **自动备份容错** - 解决可用性顾虑
3. **灵活规则配置** - 满足不同场景需求（会议室/IP/部门）
4. **资源隔离** - VIP会议不被普通会议影响
5. **本地录制管理** - 数据安全和管理便捷性

### Call-to-Action
"站点管理，让多地区部署的视频会议系统真正发挥价值。立即体验"

## 5. 视觉风格建议

- **配色方案**:
  - 主色：蓝色（科技感、专业）
  - 强调色：绿色（成功、流畅）、红色（问题、警告）
  - 背景：浅灰/白色（清爽）

- **动画风格**:
  - 流畅的过渡动画（fade, slide）
  - 数据驱动的图表动画（线条生长、数字滚动）
  - 适度使用弹性动画（spring）增加亲和力

- **字体层级**:
  - 标题：32-48px，加粗
  - 正文/旁白：24-28px，常规
  - 标注/说明：16-20px，细体
