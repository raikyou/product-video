# 站点管理功能 - 产品演示视频脚本

**总时长**: 60-75秒
**视频尺寸**: 1920x1080
**帧率**: 30 fps

---

## 场景 1: 问题场景 (0-15秒)

**视觉呈现**:
- 背景: 深色渐变 (#1a1a2e → #16213e)
- 中国地图 (china.svg) 居中显示，多个红色标记点闪烁
- 标记点之间有红色连线指向中心点（总部）

**动画效果**:
- 地图从透明淡入
- 标记点逐个弹出
- 连线从各地指向中心，带有流动光效

**文字内容**:
```
主标题: "多地区部署，资源压力山大？"
副标题 (逐行显示):
  • 视频延迟高，用户体验差
  • 总部资源吃紧，网络负载重
  • 下级会议影响总部稳定性
```

**旁白** (可选):
"多地区部署时，所有终端接入总部服务器，延迟高、压力大，下级开会还会影响总部..."

## 场景 2: 解决方案引入 (15-28秒)

**视觉呈现**:

**文字内容**:

```
功能亮点 (卡片依次弹出):
  ✓ 按规则智能分配资源
  ✓ 低延迟，高质量保障
  ✓ 会议隔离，互不干扰
```

**旁白** (可选):
"站点管理功能，让终端就近接入本地资源，开会低延迟，质量有保证！"

## 场景 3: 功能展示 - 站点列表 (28-45秒)

**视觉呈现**:
- 背景变为浅色 (#f5f7fa)
- 展示站点管理界面截图 (image-4.png)
- 界面从左侧滑入

**动画效果**:
- 界面滑入 + 淡入
- 高亮显示关键区域（站点列表、筛选功能）
- 鼠标指针动画：点击 "站点列表" → 展示数据

**文字内容**:
```
标题: "全面的站点管理"
说明文字 (依次显示):
  • 查看所有站点状态
  • 筛选区域、网络配置
  • 管理呼叫、录制、点播服务
```

**标注/高亮**:
- 用蓝色框标注 "站点名称" 列
- 用绿色框标注 "呼叫服务"、"录制服务" 列
- 用黄色框标注 "操作" 按钮

## 场景 4: 功能展示 - 规则配置 (45-60秒)

**视觉呈现**:
- 展示添加规则对话框 (image-7.png)
- 对话框从屏幕中央缩放弹出

**动画效果**:
- 对话框弹性缩放进入 (spring 动画)
- 表单字段逐个高亮
- 模拟填写效果（文字逐字输入）

**文字内容**:
```
标题: "灵活的接入规则配置"
功能说明 (气泡提示):
  → 按 IP 地址/网段分配
  → 按部门智能路由
  → 按会议室专享资源
```

**交互演示**:
- 鼠标点击 "规则类型" 选择 "终端/代理IP"
- 输入框中输入 "192.168.1.100"
- 勾选 "呼叫" 和 "点播" 应用场景
- 选择分配站点

## 动画节奏

- **快速**: 文字淡入/淡出 (15-20帧)
- **标准**: 界面滑入/弹出 (25-30帧)
- **慢速**: 地图缩放/旋转 (40-50帧)
- **转场**: 场景切换 (20-30帧)

---

## 备注

1. 所有文字使用中文，确保可读性
2. 界面截图需要添加高亮/标注效果
3. 地图动画是核心视觉元素，需要重点打磨
4. 配乐建议：轻快、科技感的背景音乐 (120-130 BPM)
5. 如需配音，语速控制在 250-280 字/分钟
